# Time Picker

Time pickers pickers provide a simple way to select a single value from a pre-determined set.

## Time Picker

![](<../../../../.gitbook/assets/image (81).png>)

```
New-UDTimePicker
```

## Locale

Specify the locale of the time picker.&#x20;

```
New-UDTimePicker -Locale fr
```
