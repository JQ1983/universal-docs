
NAME
    Remove-UATag
    
SYNOPSIS
    Removes a tag.
    
    
SYNTAX
    Remove-UATag [-Tag] <Tag> [-AppToken <String>] [-ComputerName <String>] -Id <Int64> [-UseDefaultCredentials] [<CommonParameters>]
    
    
DESCRIPTION
    Removes a tag.
    

PARAMETERS
    -AppToken <String>
        An app token to access the UA API.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -ComputerName <String>
        The HTTP address of the UA REST API server.
        
        Required?                    false
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -Tag <Tag>
        The tag to remove. Use Get-UATag to retrieve a tag object.
        
        Required?                    true
        Position?                    0
        Default value                None
        Accept pipeline input?       True (ByValue)
        Accept wildcard characters?  false
        
    -Id <Int64>
        {{ Fill Id Description }}
        
        Required?                    true
        Position?                    named
        Default value                None
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    -UseDefaultCredentials [<SwitchParameter>]
        Use default credentials when connecting to the management API
        
        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       False
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer, PipelineVariable, and OutVariable. For more information, see
        about_CommonParameters (https://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    UniversalAutomation.Tag
        
    
    
OUTPUTS
    System.Object
        
    
    
NOTES
    
    
        
    
    -------------------------- Example 1 --------------------------
    
    PS C:\> $Tag = Get-UATag -Name 'Release' 
    PS C:\> Remove-UATag -Tag $tag
    
    Removes the Release tag.
