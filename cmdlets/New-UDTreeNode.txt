
NAME
    New-UDTreeNode
    
SYNOPSIS
    Creates a tree node.
    
    
SYNTAX
    New-UDTreeNode [-Name] <String> [-Id <String>] [-Children <ScriptBlock>] [-Leaf] [-Icon <Object>] [-ExpandedIcon <Object>] [<CommonParameters>]
    
    
DESCRIPTION
    Creates a tree node. This cmdlet should be used with New-UDTreeView.
    

PARAMETERS
    -Name <String>
        The name of the node. This is displayed within the UI.
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Id <String>
        The ID of the node. This is passed to the $EventData property when the OnNodeClicked script block is set.
        
        Required?                    false
        Position?                    named
        Default value                [Guid]::NewGuid()
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Children <ScriptBlock>
        The children of this node. This should be a collection of New-UDTreeNodes.
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Leaf [<SwitchParameter>]
        This is a leaf node and should not display children.
        
        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Icon <Object>
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -ExpandedIcon <Object>
        
        Required?                    false
        Position?                    named
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer, PipelineVariable, and OutVariable. For more information, see
        about_CommonParameters (https://go.microsoft.com/fwlink/?LinkID=113216).
